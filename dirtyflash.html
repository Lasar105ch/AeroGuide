Dirty flashing lets you install a new ROM version without wiping your data or apps. This can save time and effort but needs to be done carefully.

Before You Start

  • Make a full backup (Nandroid backup via TWRP or similar recovery)
  • Charge your device to at least 60%
  • Make sure your custom recovery is updated (TWRP, OrangeFox, etc.)
  • Download the new ROM zip file and compatible GApps if needed
  • Optional: Backup important app data with apps like Titanium Backup or via ADB

Step-by-Step Dirty Flash Guide

  1. Boot into your custom recovery (usually via Power + Volume Down or use adb reboot recovery)
  2. In recovery, select "Install" and choose the new ROM zip file
  3. Important: In TWRP, uncheck "Wipe Data" or "Format Data" options; only flash the ROM zip
  4. Flash GApps if your ROM requires it
  5. Optional: Flash Magisk or other mods as needed
  6. Wipe Cache and Dalvik / ART Cache only
  7. Reboot system

Tips & Warnings

  • Dirty flashing works best when updating within the same ROM family (e.g., LineageOS 19.x to 19.x)
  • If switching ROMs or major version upgrades, perform a full wipe to avoid issues
  • Sometimes, dirty flashing causes bootloops ‚Äî keep backups handy
  • Always check the ROM's official instructions for any special steps
